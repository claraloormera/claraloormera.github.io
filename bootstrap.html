Instalación

Para instalar el bootstrap necesito un archivo html o alguna plantilla que nos ofrece bootstrap. En este caso se usó Carousel. Copié el código fuente de la plantilla en un archivo index.html y de la misma forma hice con la hoja de estilo "carousel.css". Y añadí a la carpeta del sitio las carpetas de bootstrap js, css, fonts.

Configuración de plantilla

Para personalizar la plantilla eliminé algunas cajas, edité la hoja de estilos, y adicioné clases en el html a las etiqutas necesarias siguiendo el manual de Bootstrap donde se podrán encontrar las propiedades definidas por bootstrap las cuales se usan llamándolas en el archivo html.
